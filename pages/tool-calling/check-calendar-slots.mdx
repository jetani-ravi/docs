title: "Check Calendar Slots"
description: "Learn how to integrate Voice Nova with Cal.com to fetch real-time available calendar slots during live conversations."
---

![Check Calendar Pn](/images/check-calendar.png)

| **Property** | **Description**                                                                                                                                      |
| :----------- | :--------------------------------------------------------------------------------------------------------------------------------------------------- |
| Description  | Add description for fetching the slots.<br />**Try to make it as descriptive as possible for the LLM to execute this function successfully**.        |
| API key      | You [Cal.com](http://Cal.com) API key ([**generate here**](https://app.cal.com/settings/developer/api-keys))                                         |
| Event        | You [Cal.com](http://Cal.com) event                                                                                                                  |
| Timezone     | **Select the timezone** used in your [Cal.com](http://Cal.com) event. This **helps the agent compute times accurately** based on your local setting. |
