## What is the Voice Nova API?

The Voice Nova API allows you to programmatically create, manage, and deploy intelligent voice agents for your business. Use our endpoints to automate calls, manage agent templates, retrieve analytics, and more.

<Info>
  You must have a valid Voice Nova account to generate and use APIs
</Info>

## How to create Api key

### 1. Go to the  API keys  tab in the Menu and Click on **Create API key.**

![Createapikey Pn](/images/createapikey.png)

### 2. Enter API Key name and select Expiration

![Apikeyform Pn](/images/apikeyform.png)

### 3. API Key generated

![Createdapi Pn](/images/createdapi.png)

<Info>
  The API Key will be shown only once. Hence, please save it somewhere secure.
</Info>

## Using the API Key

All API endpoints require authentication using API Key. You can generate and manage your API keys in your Voice Nova dashboard.

Include your API key in the `Authorization` header of every request:

```json
Authorization: Bearer <api_key>
```
